## ✨ Funcionalidades

- **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktops.
- **Background Interativo:** Sistema de partículas geométricas customizado usando **WebGL (OGL)** e **Shaders (GLSL)**.
- **Seção de Projetos:** Showcase interativo com carrossel dinâmico (**Swiper.js**).
- **Contato Integrado:** Formulário de e-mail funcional utilizando **EmailJS**.
- **Animações Fluidas:** Transições suaves e efeitos de entrada com **Framer Motion**.
- **Dark Mode Moderno:** Interface elegante com foco em usabilidade noturna.

## 🚀 Como Executar o Projeto Localmente

Siga os passos abaixo para rodar o projeto em sua máquina:

1. **Clonar o Repositório:**

   ```bash
   git clone https://github.com/seu-usuario/MeuPortfolio.git
   ```

2. **Instalar Dependências:**

   ```bash
   cd MeuPortfolio
   npm install
   ```

3. **Configurar Variáveis de Ambiente:**
   Crie um arquivo `.env` na raiz do projeto e adicione suas chaves do EmailJS:

   ```env
   VITE_EMAILJS_SERVICE_ID=seu_service_id
   VITE_EMAILJS_TEMPLATE_ID=seu_template_id
   VITE_EMAILJS_PUBLIC_KEY=seu_public_key
   ```

4. **Rodar em Modo de Desenvolvimento:**
   ```bash
   npm run dev
   ```
